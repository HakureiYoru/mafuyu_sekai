# Mafuyu Sekai: Neon Remake

## 操作
- 移动：W/A/S/D
- 瞄准射击：鼠标移动 + 左键
- 战术冲刺：R, 每次冲刺后在阈值时间内可以发射一次激光。
- 清屏炸弹：Space

## 更新日志
- 首页「更新日志」面板实时读取 `public/data/changelog.json`，标题会自动展示最新版本号。
- 当前版本：v2.5.0（2025-11-27）

## 更新日志维护规则
- **每次功能/平衡/资源改动都要更新** `public/data/changelog.json`，保持按时间倒序（最新在最上）。
- 版本号使用 `v主.次.修`（例如 `v2.5.1`），日期使用 `YYYY-MM-DD`。
- 每条记录至少包含：`version`、`date`、`title`（一句话摘要）、`highlights`（1~5 条要点）。
- 发布前自查：确认首页标题显示为最新版本、更新点描述与本次改动一致。
